Func 1 осуществляет бинарный поиск элемента массива по ключу и возвращает индекс найденного элемента.
Func 2 осуществляет вставку нового элемента в массив в найденную по func1 позицию, существующие элементы при этом смещаются вправо.
Если маcив пустой, новое значение ставится как первый элемент и выполнене функции заканчивается.
Если значение ключа нового элемента меньше значения ключа первого элемента, 
то новый элемент вставляется в начало массива, если больше последнего - в конец (без поиска позиции по func1).
Код функционирует, если массив формируется только через func2, в противном случаи массив должен быть
предварительно отсортирован по значению ключа по возрастанию.
